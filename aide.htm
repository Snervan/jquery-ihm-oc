Pour commencer à générer le formulaire, cliquez sur Label pour renseigner la nature du champ textuel que vous pouvez ensuite ajouter avec Zone de texte.

Une fois les deux étapes validées, vous pourrez ajouter le bouton d'envoi du formulaire avec Bouton.

Utilisez la zone de texte pour définir le texte à insérer dans le label, puis cliquez sur OK pour l'afficher dans la partie gauche de la page ou sur Annuler pour annuler l'insertion du label.

Utilisez la zone de texte pour définir l'id du champ textuel qui sera inséré, puis cliquez sur OK pour l'ajouter dans la partie gauche de la page ou sur Annuler pour annuler l'ajout de la zone de texte à gauche.

Utilisez la zone de texte pour définir le texte dans le bouton d'envoi du formulaire généré à gauche, puis cliquez sur OK pour l'ajouter dans la partie gauche de la page ou sur Annuler pour annuler son insertion.

Une fois le bouton ajouté, il ne sera plus possible de le modifier.
